1. Auto Playing Video

To present a video in HTML, video element is used with added styles to the CSS. Some attributes are added to the tag such as autoplay, muted and loop. In CSS, the video is centered at the webpage and it is adjusted by implementing the margin, padding and display properties.

2. Site-Map

A sitemap is implemented as a fundamental feature of the business website to give visitors to lead them upon finding the business on Google Maps. On Google Maps, the share button is chosen to select the Embed map. It will auto-generate a code of the location to insert in the HTML.


This feature is implemented at here. The idea of this feature was referenced from here.


3. Dropdown Menu

In HTML, a container element div class="dropdown" is used to create the dropdown content and position the dropdown content correctly with CSS.


In CSS, the .dropdown class uses position:relative, which is needed when the dropdown content to be placed right below the dropdown button by using position:absolute. The .dropdown content class holds the actual dropdown content. It is hidden by default, and will be displayed on hover. The min-width is set to 130px which is the box width of dropdown. The z-index is used to place the dropdown in front of other elements. The :hover selector is used to show the dropdown menu when the user moves the mouse over the dropdown button.


This feature is implemented at the Menu in navigation bar. The idea of this feature was referenced from here.


4. Word Hover Effect

The :hover selector is used to select elements when the visitors mouse over them. It shows the word red in colour to tell the visitors that what is the selection they are looking at.


This feature is implemented at every single page of the website at navigation bar and footer. The idea of this feature was referenced from here.


5. Box Utilities

The boxes are created to attract and give a brief introduction to the visitors about the food in the café. The colour of the boxes are alternating to give a comfortable view for the visitors.


In HTML, a grid container is created which declared as a class called grid-col-2. Besides that, grid-reversed class is the alternating position of grid-col-2.


In CSS, the grid-template-columns: repeat(2, 1fr); is used by using the repeat() notation to create a repeating pattern of tracks. The grid will consist of 1 track with 1/2 of the available space of track which basically is 2 columns. The first-child selector is used for grid-reversed and set the order to 2. This is to swap the contents and show the alternating display of boxes.
